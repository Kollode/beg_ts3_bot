I don't expect anyone else to use it, so no real docs here :)

Bot expects a 'config.json' file containing the credentials and allowed users/chatgroups:

```
{
    "ts3_server": "domain:10011",
    "ts3_password": "PW",
    "telegram_token": "TOKEN",
    "allowed_users": [1234],
    "allowed_chats": [4321]
}
```
